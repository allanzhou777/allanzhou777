A Robust Model Trained on Adversarial Attacks

In the Summer of 2022, I enrolled in an AI boot camp called Inspirit AI, a summer program taught by Stanford student researchers who've done cutting edge research. Here, I started by learning the basics of AI and ML through lectures and assignments. However, the program ended too quickly, so I contacted my camp mentor to do an individual 1 on 1 project. He agreed.

Since then, we've been working on a pipeline that can be trained on a data set through adversarially robust methods. With enough time, we hope to be able to share this with others and release our package to the public :)
